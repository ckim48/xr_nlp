About NLP Team

The objective of the NLP team was to develop a visual representation of a conversational interface for communication with virtual reality characters. This involved training the members to analyze and get valuable information from stored chat data by employing techniques such as keyword extraction and summarization.

Conversational Interface Design

Create a visual representation of a conversational interface tailored for interaction with virtual reality characters.

Chat Data Visualization

Explore and leverage various visualization tools to enhance the presentation of insights gleaned from chat data.

NLP Analytical Techniques

Master fundamental analytical methods, such as keyword extraction and summarization, to derive valuable insights from the stored conversations.

Features

The NLP Team has created a web-based dashboard which has two core functionalities: Dialogue Analysis and Trend Analysis.

Dialogue Summarization

Utilizes the GPT-3.5 model to summarize the input dialogue, providing a concise summary in 5 sentences.

Sentiment Analysis

Analyzes the sentiment of the input dialogue, providing three sentiment words and explanations.

Keyword Extraction

Extracts the top 10 keywords from the dialogue using the KeyBERT model.
